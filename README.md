

  # ASSOS – Application de gestion de tontine

**ASSOS** est une application web innovante dédiée à la gestion des associations et des tontines. Elle facilite l’organisation, la gestion des membres, des cotisations et la tenue des activités courantes.  
Ce projet est développé principalement en TypeScript, CSS et HTML.

## Fonctionnalités principales

- Gestion des membres d’association
- Suivi et gestion des cotisations
- Comptabilité simplifiée
- Historique des transactions
- Tableaux de bord et statistiques
- Interface responsive et moderne

## Démarrer le projet

### Prérequis

- [Node.js](https://nodejs.org/) (version recommandée : >= 16)
- Un gestionnaire de paquets (npm ou yarn)

### Installation

1. Clonez ce dépôt :

   ```bash
   git clone https://github.com/kaezerwatto/ASSOS.git
   cd ASSOS
   ```

2. Installez les dépendances :

   ```bash
   npm install
   # ou
   yarn install
   ```

3. Lancez le serveur de développement :

   ```bash
   npm run dev
   # ou
   yarn dev
   ```

L’application sera accessible à l’adresse affichée dans le terminal (par défaut http://localhost:3000).
